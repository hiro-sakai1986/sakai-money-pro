# サカイ家 MONEY Pro 6.0 第二弾

## 実装内容
- iPhone向けホーム画面
- 預金・投資・年間配当・毎月積立の集計
- 本人・夫の保有資産と積立を完全分離
- 保有数・取得単価・現在価格から評価額と損益を自動計算
- 入力日・購入開始日・年間配当
- 積立月数・累計入金額を自動計算
- 保有資産と積立の編集・削除
- 銘柄検索・並べ替え
- 資産配分ドーナツグラフ
- JSONバックアップ・復元
- 任意のパスワードロック
- パスワード設定時はAES-GCMでブラウザ保存データを暗号化
- 10分間操作がない場合の自動ロック
- ダークモード
- PWA・オフライン対応

## データ移行
第一弾の `sakaiMoneyPro6FirstRelease` データが同じブラウザに残っている場合、初回起動時に第二弾へ自動移行します。

## 重要
パスワードを忘れると暗号化データを復旧できません。設定後は、設定画面からバックアップJSONも保存してください。
