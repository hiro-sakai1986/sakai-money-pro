# サカイ家 MONEY Pro 5.3 第一弾

## 追加した機能
- 本人・夫の保有資産を完全分離
- 本人・夫の積立・NISAを完全分離
- 保有資産に入力日を追加
- 積立に購入開始日を追加
- 保有数・取得単価・現在価格から評価額と損益を自動計算
- 積立開始日から積立月数と累計入金額を自動計算
- 保有資産と積立の削除機能
- 5.2のAES-GCM暗号化と暗号化バックアップを維持
- 既存の5.2データを自動移行

## 更新方法
GitHubリポジトリで、このZIP内のファイルをすべて上書きアップロードしてください。
アップロード後、Safariでページを再読み込みしてください。古い表示が残る場合は、Safariを閉じて再度開いてください。

## 大切な注意
更新前に作成した暗号化バックアップは、そのまま復元できます。念のためバックアップファイルは削除せず保管してください。
