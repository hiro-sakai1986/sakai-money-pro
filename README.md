# サカイ家 MONEY Pro

サカイ家専用の家計・資産管理アプリです。

## 主な機能
- 家計簿（月別保存・年間集計）
- 夫婦の投資・配当金
- 住宅ローン
- 保険・教育費
- 未来予測
- バックアップ書き出し・読み込み

## GitHub Pages 公開
Settings → Pages → Deploy from a branch  
Branch: main / Folder: /(root)

公開URL:
https://hiro-sakai1986.github.io/sakai-money-pro/

## 大切なこと
家計データはSafariのブラウザ内に保存されます。
公開URLが同じなら継続利用できます。
更新前にアプリ内の「バックアップを書き出す」を使うと安心です。
