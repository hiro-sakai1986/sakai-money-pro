# サカイ家 MONEY Pro 8.0 Alpha

7.0 第三弾Dの全データを引き継ぎながら、8.0用の新しい画面デザインと保存領域へ移行した最初の開発版です。

## 8.0 Alphaで変更したこと
- iPhoneアプリ風の新しいホームデザイン
- 8.0専用ストレージキー（7.0 第三弾Dから自動移行）
- ライフプランタブの名称整理
- 日付・端末内保存ステータス表示
- PWAキャッシュ更新

既存の家計・投資・住宅ローン・教育資金・目的別積立・ライフイベント機能はそのまま利用できます。


## 8.0 Alpha データ復旧修正版

旧バージョンで実際に使われていた保存名も検索し、8.0が先に空データを保存した後でも、
同じブラウザー内に残っている旧データを自動で引き継ぐように修正しました。


## 8.0 Beta 家計画面

- 月別の収入・支出・収支・貯蓄率を表示
- 月ごとの支出予算と予算オーバー判定
- よく使うカテゴリから簡単入力
- 家計記録の編集・削除
- 6か月の収入・支出グラフ
- 支出カテゴリ別の内訳と割合
- 8.0 Alphaの保存データをそのまま引き継ぐため、保存キーは維持


## 8.0 Beta 第2弾 投資画面

- 本人・夫・家族合計の投資ダッシュボード
- 投資評価額・元本・損益率・年間配当・毎月積立を一画面に集約
- 資産種類別と口座別の構成比
- 集中度、価格未入力、積立額、配当月未入力の端末内チェック
- 保有資産・積立・ランキング・配当をタブで切替
- 保有商品の構成比バー、NISA口座チップ、入力不足表示
- 既存の8.0 Betaデータをそのまま維持（保存キー変更なし）


## 自動保存強化修正版

- 保存名を今後の更新でも固定
- 通常保存に加えて予備保存（IndexedDB）を追加
- アプリを閉じる・バックグラウンドへ移す直前にも保存
- 通常保存が消えた場合は予備保存から自動復旧
- GitHub Pages更新後も同じ保存領域を継続使用
